# 抖音视频下载器 Pro

一个功能强大的抖音视频批量下载工具,支持用户主页视频批量下载和单个视频下载,具有现代化的图形界面和完善的用户管理系统。

## 功能特性

### 核心功能
- **批量下载**: 支持下载抖音用户主页的所有视频
- **单视频下载**: 支持下载单个抖音视频
- **分页下载**: 可设置起始页码和下载数量
- **自动命名**: 自动使用视频标题作为文件名
- **进度显示**: 实时显示下载进度和状态

### 用户管理
- **用户注册**: 支持新用户注册账号
- **用户登录**: 登录验证机制
- **记住密码**: 可选择记住登录信息
- **密码加密**: 使用Base64加密存储密码

### 界面功能
- **现代化UI**: 采用渐变色和现代化设计风格
- **多页面导航**: 包含多个功能页面
  - 批量下载页面
  - 视频管理页面
  - 文件夹浏览页面
  - 下载统计页面
  - 偏好设置页面
  - 软件交流区
  - 运行日志页面
  - 帮助中心
  - 关于页面

### 视频管理
- **视频列表**: 查看已下载的所有视频
- **文件管理**: 删除、打开视频文件
- **统计信息**: 查看视频数量、文件大小等统计
- **文件夹浏览**: 快速打开视频保存文件夹

## 技术栈

- **Python 3.x**: 主要编程语言
- **PyQt5**: 图形用户界面框架
- **DrissionPage**: 浏览器自动化工具
- **requests**: HTTP请求库
- **Chromium**: 浏览器引擎

## 使用说明

### 首次使用

1. **注册账号**: 首次使用需要注册账号
   - 点击"注册"按钮
   - 输入用户名和密码
   - 确认密码后完成注册

2. **登录系统**: 使用注册的账号登录
   - 输入用户名和密码
   - 可选择"记住密码"
   - 点击"登录"进入主界面

### 批量下载

1. **获取用户主页URL**: 
   - 打开抖音网页版
   - 进入目标用户的主页
   - 复制URL链接(格式: `https://www.douyin.com/user/...`)

2. **设置下载参数**:
   - 在"抖音主页URL"输入框粘贴链接
   - 选择起始页码(默认第1页)
   - 选择下载数量(可选择全部或指定数量)

3. **开始下载**:
   - 点击"批量下载"按钮
   - 等待下载完成
   - 可随时点击"停止下载"中断

### 单视频下载

1. **获取视频URL**:
   - 打开抖音视频页面
   - 复制视频链接(格式: `https://www.douyin.com/video/...`)

2. **下载视频**:
   - 在"单视频URL"输入框粘贴链接
   - 点击"下载单视频"按钮
   - 等待下载完成

### 视频管理

- **查看视频**: 在"视频管理"页面查看所有已下载视频
- **删除视频**: 选中视频后点击"删除选中"
- **打开视频**: 双击视频或选中后点击"打开文件"
- **刷新列表**: 点击"刷新列表"更新视频列表

### 文件夹浏览

- **查看路径**: 显示当前视频保存路径
- **打开文件夹**: 直接打开视频保存文件夹
- **查看统计**: 显示视频数量和占用空间

### 下载统计

- **总视频数**: 已下载的视频总数
- **总大小**: 所有视频文件的总大小
- **成功率**: 下载成功的百分比
- **下载次数**: 执行下载操作的次数

### 偏好设置

- **修改保存路径**: 可更改视频默认保存位置
- **刷新重置**: 重置所有设置参数

## 注意事项

1. **网络连接**: 确保网络连接稳定,下载大视频时可能需要较长时间
2. **浏览器要求**: 需要安装Chromium浏览器(DrissionPage会自动管理)
3. **存储空间**: 确保有足够的磁盘空间存储视频
4. **合法使用**: 请遵守抖音平台的使用条款,仅下载有权限的视频内容
5. **账号安全**: 妥善保管账号密码,不要泄露给他人

## 常见问题

### Q: 下载失败怎么办?
A: 
- 检查网络连接是否正常
- 确认URL链接是否正确
- 尝试重新下载或更换网络环境

### Q: 视频无法播放?
A: 
- 确认视频文件是否完整下载
- 检查文件大小是否正常
- 尝试使用其他播放器打开

### Q: 登录失败?
A: 
- 确认用户名和密码是否正确
- 检查是否已注册账号
- 尝试重新注册新账号

### Q: 下载速度慢?
A: 
- 检查网络带宽
- 尝试在网络空闲时段下载
- 减少同时下载的视频数量
